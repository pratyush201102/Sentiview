# Sentiview MVP Scope (Week 1–2 Finalized)

## In Scope (Current Build)
- Backend API for keyword-based Reddit collection.
- VADER-based sentiment classification.
- PostgreSQL persistence for searches and sentiment results.
- CSV export and searchable history endpoints.
- Documentation for architecture and local setup.

## Out of Scope (Later Phases)
- Frontend React dashboard implementation.
- Multi-source ingestion beyond Reddit.
- Advanced model fine-tuning and domain adaptation.
- Authentication/authorization and user accounts.

## Success Criteria
- API accepts keyword and returns sentiment-tagged records.
- Sentiment counts and history are persisted and retrievable.
- Data is exportable via CSV endpoint.
- Setup can run locally with minimal configuration.
